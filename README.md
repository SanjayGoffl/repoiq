## 🚀 Overview

**RepoIQ** is an AI-powered platform designed to demystify any GitHub repository in seconds. Simply paste a repository link, and RepoIQ transforms the raw codebase into an interactive, conversational learning experience. Whether you're exploring a new stack, trying to understand a complex project, or looking for structured learning, RepoIQ works as your personal, hyper-personalized tutor.

---

## ✨ Key Features

### 🔍 Instant Repository Analysis
* **Stack Insights:** Automatically detects the frameworks, languages, and technologies used.
* **Codebase Meta:** Gets the lines of code and complete dependency tree.
* **System Requirements:** Estimates the approximate RAM and resources needed to run the project.

### 📂 Intelligent Folder Visualization
* **Structure Map:** Visually lays out the project's folder hierarchy.
* **Smart Prioritization:** Highlights the most crucial files and explains *why* they are important to the overall architecture.

### 💻 Interactive Code Explorer
* **Deep Dive Code Reader:** Click any file to open an interactive code viewer.
* **Line-by-Line Context:** Select any line or specific variable to see an instant explanation of its purpose and the library it belongs to.
* **Ask the AI:** Have a specific doubt? Chat directly with the AI about that exact block of code.

### 📚 Guided Lesson Mode
* **Step-by-Step Learning:** The AI breaks down the entire repository into logical, bite-sized lessons.
* **Conversational Teaching:** The AI guides you through each lesson interactively, ensuring you understand the core mechanics before moving forward.

### 🧠 Persistent `memory.md`
* **Personalized Teaching:** RepoIQ remembers who you are, how you learn, and your past interactions.
* **Continuous Optimization:** The AI builds a `memory.md` profile of your learning style, optimizing and adapting its explanations over time to suit your unique needs.

## 🚀 Getting Started

1. **Clone the project:**
   ```bash
   git clone https://github.com/your-username/repoiq.git
   cd repoiq
   ```
2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   # or 
   pnpm install
   ```
3. **Set up Environment Variables:**
   Copy the `.env.example` to `.env.local` and add your required API keys.
   ```bash
   cp .env.example .env.local
   ```
4. **Run the Development Server:**
   ```bash
   npm run dev
   ```
5. **Explore:** Open [http://localhost:3000](http://localhost:3000) and paste your first repository link!
